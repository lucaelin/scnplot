x32 / m32 Scene-Plot

This browser is outdated or does not support features required for this tool to run. Please update or use another browser (Firefox or Chrome recommended).

This tool can be used to get a quick overview of x32 and m32 scene configurations. Just select or drop a valid ".scn"-file using the button at the top of this page.

No ".scn"-file will ever be transmitted anywhere - analysis and plotting is done locally.

A copy of this tool can be obtained at GitHub.com.
